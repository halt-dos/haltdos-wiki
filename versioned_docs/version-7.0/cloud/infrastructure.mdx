# Infrastructure

Haltdos Cloud leverages multiple Cloud Service Providers (CSP) as backend infrastructure partner. By leveraging multiple CSPs, Haltdos provide redundancy, reliability, compliance and data localization benefits to our customers.
Haltdos Cloud ensures that:

- At any point in time, there are at least two or more CSPs through which traffic can be routed for any website protected behind Haltdos Cloud.
- High uptime of 99.999 (backed by SLA)
- Isolating traffic for multiple customers
- Only thoses PoPs are from our CSP partners are chosen which are closer to origin (location where web appliction server is hosted).
- Metadata (statistical information and logs) residency in the same country as the origin
- DDoS attack mitigation at source of attack

## FAQs

##### **Do customer have choice in choosing CSP ?**

Yes, we do try our best honor customer's choice of CSP. However, in the event a CSP is down, Haltdos Cloud Platform will automatically switch over to other available and running CSPs.

##### **Does my traffic transends across countries ?**

We take data residency and data localization very seriously. For a country like India with multiple PoP locations and multiple CSPs, traffic and data resides in India and does not transends country borders. However, for a country like Portugal, due to lack of adequate presence of multiple CSPs, The data residency and locatization is restricted within EU boundaries.
