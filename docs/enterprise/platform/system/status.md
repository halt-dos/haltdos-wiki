# Status

---

This Shows the status of the current Selected stack

![Analytics](/img/platform/v8/docs/sysStatus.png)

### **Other Features**  

#### **Summary**  

Shows the summary of the stack.  

![Analytics](/img/platform/v8/docs/sys_summary.png)

#### **Measurements**

Shows the measurement of the selected Stacks, i.e, CPU, RAM and DISK.

![Analytics](/img/platform/v8/docs/sys_measurement.png)

#### **Service Status**

Shows the status of the service. 

![Analytics](/img/platform/v8/docs/service_status.png)
